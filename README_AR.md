# 🚀 FarmTech - المشروع جاهز للتشغيل!

تم تحديث المشروع ليعمل مباشرة على الويندوز بدون Docker، وتم تبسيط نظام التشغيل.

## ⚡ ابدأ بسرعة

لقد أنشأت لك ملف تشغيل واحد للويندوز:

### 1️⃣ تشغيل المشروع:
قم بالضغط مرتين على:
```bash
Run_FarmTech.bat
```
- سيقوم بفتح نافذتين (واحدة للباك-إند وواحدة للفرونت-إند).
- تأكد من تثبيت مكتبات Python في البيئة الافتراضية `.venv` وتثبيت مكتبات Node.js (`npm install`).

---

## 🌐 روابط الوصول

| الخدمة | الرابط | الوصف |
|------|--------|--------|
| **الفرونت إند (Next.js)** | http://localhost:3000 | واجهة المستخدم |
| **الباك إند (Django)** | http://127.0.0.1:8000 | واجهة API |
| **لوحة التحكم (Admin)** | http://127.0.0.1:8000/admin/ | لوحة التحكم |

### 🔐 بيانات الدخول الافتراضية (إذا كانت قاعدة البيانات جاهزة):
```
البريد: admin@farmtech.local
كلمة السر: admin123
```

## 🛠️ المتطلبات التشغيلية (يدوي):
إذا أردت تشغيل كل جزء يدوياً:

### الباك إند:
```bash
cd farmtech_fixed/backend
..\..\.venv\Scripts\activate
python manage.py runserver
```

### الفرونت إند:
```bash
cd farmtech_fixed/frontend
npm run dev
```
